- This is a fun project combining Arduino and Unity.
In this project, we recreate the classic Nintendo duck shooting game from the 90s.
In this game, we design a gun with a light sensor to detect flying ducks on the screen.
Arduino programming is used to send signals to the PC.
Unity is used to program the duck shooting game and connect it to the gun via a USB connection to Arduino.
- How can I run this project?
The source code is divided into two parts for Arduino and Unity. Along with that is a file describing the circuit diagram connecting the electronic components needed.
You can download the source code and use Arduino Studio to view and run the Arduino commands.
Use Unity to open the pre-made projects in the Unity folder to test the game part.
For more information on how to use this project, you can follow additional videos on the Khoa Hoc Vui channel.
https://www.youtube.com/watch?v=W93gyR3Oa1Q

- Đây là dự án vui kết hợp giữa Arduino và Unity.
+ Trong dự án này chúng ta cùng dựng lại trò chơi bắn vịt của Nintendo ở những năm thập niên 90.
